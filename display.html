"Display" xossasi

Block element

Block element - ota-ona (parent) elementni to'liq kengligiga yoyiladi va har doim yangi qatordan boshlanadi. div, h1, h2, ... h6, ul, li va yana bir qator taglar block element yaratad.

Inline element

Inline element - faqatgina o'zini kontenti uchun yetarli bo'ladigan kenglikga ega bo'ladi va yangi qatordan boshlanmaydi, aksincha yonma-yon joylashadi. span, a va yana bir qator taglar inline element yaratadi.

Display

Display - elementimizni veb sahifamizda qanday ko'rsatilishini belgilaydi

  • display: block; - element block elementga aylanadi
  • display: inline; - element inline elementga aylanadi
  • display: none; - elementni veb sahifasida ko'rsatilmasligini ta'minlaydi
  • inline-block - element inline elementga aylanadi, lekin block elementning xususiyatlariga ega bo'ladi

Text-align, Vertical Align va Float xossalari

Text-align

Text-align - element ichidagi matnni gorizontal ravishda hizalaydi

  • Left - matnni chap tarafga tekislaydi
  • Center - matnni markazga tekislaydi
  • Right - matnni o'ng tarafga tekislaydi
  • Justify - matnni ikki tomonlama tekislaydi
Vertical align

Vertical align - element ichidagi matnni vertikal ravishda hizalaydi

/* Misol */ .block { display: block; text-align: center; } .inline { display: inline; } .none { display: none; }
